安装 Hugo

本文以 macOS 系统为例，安装 Hugo，其他系统可参考 Hugo 安装指南。

HomeBrew 是 macOS 上的包管理器，我们将用它来安装 Hugo。当然也可以用源码编译的方式安装，这里是安装指引。

brew install hugo

验证你的安装。

hugo version
